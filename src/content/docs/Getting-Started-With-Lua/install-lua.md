# Install Lua

The official Lua installation instructions are found [here.](https://www.lua.org/manual/5.5/readme.html)

# Lua Releases and 5.5

Lua doesn't release updates often. This allows the language to be more stable. Despite this, the 5.5 release had a number of [breaking changes](https://www.lua.org/manual/5.5/manual.html#8).

This guide will use the latest major version of Lua, 5.5
